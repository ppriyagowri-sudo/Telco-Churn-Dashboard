# 📊 Telco Churn Dashboard

## 📌 Project Overview
This project presents a Power BI dashboard built to analyze customer churn in a telecom company. It identifies patterns, trends, and key factors influencing customer retention, enabling stakeholders to drill down by segment, region, and service type for targeted retention campaigns.

## 📁 Files Included
- `Telco_Churn_Dashboard.pbix` → Power BI dashboard file
- `Telco_Churn_Data.xlsx` → Dataset used for analysis

## 🎯 Objectives
- Analyze customer churn behavior
- Identify high-risk customer segments
- Understand key factors affecting churn (contract type, tenure, monthly charges)
- Provide insights for better retention decision-making

## 📊 Key Insights
- Churn rate by contract type
- Impact of monthly charges on churn
- Customer tenure vs. churn
- Services used by churned customers
- Segment and region-level churn drill-down

## 🛠️ Tools & Technologies
- Power BI (DAX, Power Query, interactive dashboards)
- Microsoft Excel (data preparation)

## 🚀 How to Use
1. Clone the repository:
   ```
   git clone https://github.com/ppriyagowri-sudo/Telco-Churn-Dashboard.git
   ```

3. Explore the dashboard visuals and insights
