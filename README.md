# 📊 Telco Churn Dashboard

## 📌 Project Overview
This project presents a **Power BI dashboard** built to analyze customer churn in a telecom company. It helps identify patterns, trends, and key factors influencing customer retention.

---

## 📁 Files Included
- `Telco_Churn_Dashboard.pbix` → Power BI dashboard file  
- `Telco_Churn_Data.xlsx` → Dataset used for analysis  

---

## 🎯 Objectives
- Analyze customer churn behavior  
- Identify high-risk customers  
- Understand factors affecting churn  
- Provide insights for better decision-making  

---

## 📊 Key Insights
- Churn rate based on contract type  
- Impact of monthly charges on churn  
- Customer tenure vs churn  
- Services used by churned customers  

---

## 🛠️ Tools & Technologies
- Power BI  
- Microsoft Excel

---

## 🚀 How to Use
1. Download the `.pbix` file  
2. Open it using Power BI Desktop  
3. Explore the dashboard visuals and insights
